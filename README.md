# ☀️ SunCart – Summer Essentials Store

SunCart is a modern summer eCommerce web application where users can explore and purchase seasonal products such as sunglasses, summer outfits, skincare items, beach accessories, and more.



## 🎯 Project Purpose

The purpose of this project is to build a responsive modern eCommerce platform using Next.js with authentication, protected routes, profile management, and dynamic product browsing.

---

## ✨ Key Features

- Responsive design for Mobile, Tablet, Desktop
- Modern Navbar with Login/Register system
- Google Authentication
- User avatar and logout system
- Protected Product Details page
- Product data from JSON file
- Popular Products section
- Summer Care Tips section
- Top Brands section
- My Profile page
- Update Profile information
- Toast notifications
- Smooth UI animations using React Spring
- Persistent Navbar and Footer

---

## 🔐 Authentication Features

- Login with Email & Password
- Google Login
- Register New User
- Protected Routes
- Logout Functionality

---

## 📦 Technologies Used

- Next.js
- React.js
- Tailwind CSS
- NextAuth
- React Hot Toast
- React Icons
- React Spring
